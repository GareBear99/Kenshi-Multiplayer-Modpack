# Contributing

Thanks for helping improve the Kenshi Multiplayer Modpack repository.

## What contributions are most useful right now

- better install documentation
- exact mod source links
- compatibility notes
- load order validation
- bug reports with reproducible steps
- multiplayer-safe mod recommendations

## Before opening a pull request

Please include:
- what changed
- why it changed
- whether it affects compatibility
- whether it changes the tested mod list
- whether it changes install steps

## Good contribution rules

- keep claims specific
- do not mark a mod “safe” without testing notes
- do not silently replace third-party files
- do not add giant bundled archives without documenting source and purpose
- prefer small, auditable changes
